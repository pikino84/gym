php artisan make:controller -mcr
-m: Especifica que el controlador debe estar vinculado a un modelo. 
    Laravel generará automáticamente los métodos CRUD básicos en el controlador para interactuar con el modelo correspondiente.

-c: Genera el controlador con los métodos de recurso RESTful. Estos métodos incluyen index, create, store, show, edit, update y destroy.

-r: Genera también un archivo de rutas (routes/web.php) para el controlador, con las rutas de recurso predefinidas 
    correspondientes a los métodos del controlador.
